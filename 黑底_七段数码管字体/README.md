# PPT 倒计时器 (PPT Countdown Timer)

这是一个专为 PPT 演讲设计的倒计时器，支持始终置顶显示，即使 PPT 全屏播放时也能看到。

## 功能特点

- **始终置顶**：窗口浮动在所有窗口（包括全屏 PPT）之上。
- **透明背景**：完全透明的背景，不影响 PPT 演示效果，文字带有阴影确保清晰可见。
- **酷炫界面**：
  - 7段数码管字体显示
  - 动态发光效果
  - 进度条实时显示剩余时间
  - 数字颜色随时间变化（绿色→黄色→红色）
- **预设时间**：1分钟、3分钟、5分钟、8分钟、15分钟快速选择。
- **自定义时间**：支持输入任意分钟数。
- **清晰显示**：巨大数字，适合演讲者查看。
- **控制按钮**：开始、暂停、重置。
- **智能提醒**：
  - 时间剩余 >50%：绿色显示
  - 时间剩余 20-50%：黄色警告
  - 时间剩余 <20%：红色危险提示
  - 时间结束：全屏提醒 + 滴滴声音
- **可拖拽**：按住窗口空白处即可拖动位置。

## 安装与运行

确保已安装 [Node.js](https://nodejs.org/)。

1. 打开终端（命令提示符/PowerShell），进入本目录：
   ```bash
   cd c:\你的项目目录
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动应用：
   ```bash
   npm start
   ```

## 打包为 EXE 文件

使用 electron-builder 打包为 Windows 安装程序：

```bash
npm run build
```

打包完成后，在 `dist` 目录中会生成安装程序。

**注意**：打包后的应用体积约 200MB，这是 Electron 应用的正常体积（包含完整的 Chromium 浏览器 + Node.js 运行时）。

## 文件说明

- `main.js`: Electron 主进程，负责创建置顶窗口和全屏提醒窗口。
- `renderer.js`: 计时器逻辑实现，包含进度条和颜色变化。
- `style.css`: 界面样式，包含发光效果和动画。
- `index.html`: 主界面。
- `alert.html`: 全屏提醒界面。
- `didi.wav`: 提示音文件。
- `7-segments/`: 7段数码管字体文件夹。
- `package.json`: 项目配置和打包设置。

## 技术栈

- **Electron**: 桌面应用框架
- **HTML/CSS/JavaScript**: 前端技术
- **7-Segment Font**: 数码管字体
- **Web Audio API**: 音频播放

## 系统要求

- Windows 10/11
- Node.js 14+ (开发环境)
