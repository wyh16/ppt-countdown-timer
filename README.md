# PPT 倒计时器 (PPT Countdown Timer)

这是一个专为 PPT 演讲设计的倒计时器，支持始终置顶显示，即使 PPT 全屏播放时也能看到。

## 项目简介

本项目提供两个版本的 PPT 倒计时器，满足不同使用场景的需求：

### 版本说明

#### 1. 黑底_七段数码管字体版本（推荐）

**特点**：
- 🎨 **酷炫界面**：黑色背景 + 7段数码管字体
- ✨ **动态发光效果**：数字带有霓虹灯般的发光效果
- 📊 **进度条显示**：实时显示剩余时间百分比
- 🎯 **智能颜色变化**：
  - 绿色（>50%）：时间充裕
  - 黄色（20-50%）：时间紧张
  - 红色（<20%）：时间紧迫
- 🔊 **全屏提醒**：时间结束时全屏显示 + 滴滴声音
- 🚀 **现代化设计**：适合追求视觉效果的用户

**适合场景**：
- 需要视觉冲击力的演讲
- 追求酷炫效果的展示
- 大屏幕投影环境

#### 2. 透明版本

**特点**：
- 🪟 **完全透明背景**：不影响 PPT 演示效果
- 🎯 **简洁设计**：专注于功能，界面简洁
- 🔊 **声音提醒**：时间结束时播放提示音

**适合场景**：
- 需要看到背景内容的演示
- 追求简洁实用的用户
- 小型会议室环境

## 快速开始

### 安装与运行

1. **克隆仓库**：
   ```bash
   git clone https://github.com/你的用户名/ppt-countdown-timer.git
   cd ppt-countdown-timer
   ```

2. **选择版本**：
   - 黑底版本：进入 `黑底_七段数码管字体` 文件夹
   - 透明版本：进入 `透明版` 文件夹

3. **安装依赖**：
   ```bash
   npm install
   ```

4. **启动应用**：
   ```bash
   npm start
   ```

### 打包为 EXE 文件

```bash
npm run build
```

打包完成后，在 `dist` 目录中会生成安装程序。

## 功能特点

### 通用功能
- ✅ **始终置顶**：窗口浮动在所有窗口（包括全屏 PPT）之上
- ✅ **预设时间**：1分钟、3分钟、5分钟、8分钟、15分钟快速选择
- ✅ **自定义时间**：支持输入任意分钟数
- ✅ **控制按钮**：开始、暂停、重置
- ✅ **可拖拽**：按住窗口空白处即可拖动位置
- ✅ **声音提醒**：时间结束时播放提示音

### 黑底版本独有功能
- ✨ 7段数码管字体显示
- ✨ 动态发光效果
- ✨ 进度条实时显示
- ✨ 数字颜色随时间变化
- ✨ 全屏提醒界面

## 技术栈

- **Electron**: 桌面应用框架
- **HTML/CSS/JavaScript**: 前端技术
- **7-Segment Font**: 数码管字体（黑底版本）
- **Web Audio API**: 音频播放

## 系统要求

- Windows 10/11
- Node.js 14+ (开发环境)

## 使用建议

- **演讲比赛**：推荐使用黑底版本，视觉效果更佳
- **日常会议**：推荐使用透明版本，简洁实用
- **大型展示**：推荐使用黑底版本，全屏提醒更醒目
- **小型讨论**：推荐使用透明版本，不遮挡内容

## 注意事项

- 打包后的应用体积约 200MB，这是 Electron 应用的正常体积（包含完整的 Chromium 浏览器 + Node.js 运行时）
- 首次运行需要安装依赖，请确保网络连接正常
- 如需修改提示音，可替换 `didi.wav` 文件
